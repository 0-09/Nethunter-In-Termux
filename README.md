# Nethunter-In-Termux
This is a script by which you can install Kali nethunter (Kali Linux) in your termux application without rooted phone 
### Steps
1. Download scriptÂ in **HOME** `curl -LO https://raw.githubusercontent.com/Hax4us/Nethunter-In-Termux/master/kalinethunter`
2. Give execution permission `chmod +x kalinethunter`
3. Run script `./kalinethunter`
4. Now just start kali nethunter `startkali`
